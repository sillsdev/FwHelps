408 Computer Technology

Description

This category is used to index information on computer technology including information on hardware, software (i.e., programs, applications), maintenance and repair, application and use (e.g., word processing, accounting, databases, communication); public and private accessibility to services; specialized personnel (e.g., programmers); special commercial services; etc.

This category was introduced in 1997.
