1210 Archaeological Excavation Methods

Description

This category is used to index statements on archaeological excavation methods (e.g., Wheeler-Kenyon method, block method); what type of tools were used during excavation; whether strata were excavated in natural or arbitrary levels; whether the soil was screened, and if so, what size mesh was used; whether soil samples were taken for specialized analysis, etc.

This term was introduced in 1997. The description was rewritten completely in 2000.
