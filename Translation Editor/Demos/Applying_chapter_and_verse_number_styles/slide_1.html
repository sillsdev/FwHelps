Applying chapter and verse number styles

Translation Editor is a word processing program that “knows about chapters and verses.”

If you type chapter and verse numbers with the vernacular text instead of inserting them,
you will need to apply character styles to identify and format the numbers.

If you translate part of a chapter not beginning at verse one, you must type the numbers.
